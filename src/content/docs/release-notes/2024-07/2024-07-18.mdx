### ✔️팝업 이미지 반응형 업데이트
<Figure src={image} caption={"팝업 이미지"} />
팝업 이미지의 해상도가 접속한 환경에 따라 조정되도록 반응형으로 변경되었어요.\

### ✔️웹 취소 데이터 이력 저장 방식 변경
웹 취소 데이터의 이력 저장 방식이 변경되었어요.\
기존에는 결과와 상관없이 데이터 이력을 저장했지만, 이제는 정상적인 결과를 가진 데이터 이력만 저장하게 됩니다.
