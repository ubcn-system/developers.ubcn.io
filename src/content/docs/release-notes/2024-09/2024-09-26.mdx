    ## ✔️VMMS 정기 업데이트(예정)
    {/*VMMS 의 정기 업데이트가 진행되었어요.\
    이번 주 업데이트 된 내역은 아래와 같습니다.*/}

    ### 1. 세션 유지 방식 변경 (재반영)
    + 세션의 유지 방식이 기존 `자동 로그아웃`에서 `시간 내 동작 시 세션 시간 연장`으로 변경되었어요.
        + 기존에는 일정 시간 동안 사용자의 동작이 없을 경우 자동 로그아웃 되었는데, 이제는 사용자의 동작이 있을 경우 세션 시간이 연장됩니다.
        + 이를 통해 사용자가 더 편리하게 서비스를 이용할 수 있도록 개선했어요.

    ### 2. 거래내역 조회 기간 달력 UI 변경 (재반영)
    <Figure src={Image1} caption={"거래내역 페이지"} width={700}/>
    + 조회 기간 검색 내 달력 UI 가 변경되었어요.
        + 기존에는 달력 내 일부 이미지가 보이지 않거나 미래 일자에 대한 선택이 가능한 문제가 있었어요.\
        이런 부분을 개선하고, 보다 더 사용하기 쉽고 보기 편한 새로운 달력 UI 로 변경했어요.
        + 나머지 페이지들도 순차적으로 업데이트 할 계획이에요.

    ### 3. 기타 버그 수정 (재반영)
    + 로딩 관련 기타 버그 수정 사항이 있어요.
    + 운영자판기 관리 페이지에서 검색필드를 추가해도 '소속을 선택하세요' 문구가 나오는 부분이 불편하다는 의견을 수렴해 수정했어요.
        + 소속을 선택하지 않아도 검색이 가능해요. (검색 시 소속을 선택하지 않으면 전체 소속에서 검색됩니다.)
    + 미사용 스타일 CSS 가 제거되었어요.
    + 조회 기간에 입력 가능한 자릿수 제한을 추가했어요.
    + 카드사 심사현황의 `템플릿`을 변경했어요. (서식의 변경)

    ### 4. 사용자 권한 표기 추가
    <Figure src={Image2} caption={"사이드바"} width={300}/>
    - VMMS 페이지의 좌측에 계정에 할당된 권한 정보가 추가 표기했어요.
        - 권한 정보는 계정에 할당된 권한을 표기하며, 해당 권한에 따라 페이지 접근이 제한될 수 있어요.

    ### 5. VAN 가맹점의 'EDC입금' 내용 추가
    <Figure src={Image3} caption={"VAN 가맹점의 EDC 거래 매입거절 건"} width={300}/>
    - VAN 가맹점은 각 카드사로부터 직접 가맹점의 매출을 입금받아요.
    - 일부 카드사에서는 가맹점의 부담을 줄이기 위해 VAN 사로 입금 정보를 전달하지 않는 대신 별도 수수료를 부과하지 않아요.
    - VMMS 에서는 VAN사로부터 입금 결과를 수신받지 못해 기존에는 `매입거절` 로 표기하고 있었는데, VAN 사에서 `EDC입금` 이라는 상태로 문구를 변경함에 따라
    함께 변경하였어요.
    - 카드사에서 직접 입금한 것인지 확인할 수 있도록 했어요.
    - EDC 특약 가맹점 여부를 확인할 수 있게 되었어요.
    - `EDC입금`상태를 조회하시려면 `매입거절`을 체크 후 조회해주세요.

    ### 6. 서비스 코인 (2024.10.07~2024.10.31)
    + 서비스 코인에 대한 API 기능이 추가 예정이에요.
    + 이에 따른 권한 기능이 추가되었어요.
    + 서비스 코인을 사용할 수 있는 계정을 구분할 수 있도록 권한 기능이 개발될 예정이에요.
        + 권한이 있는 사용자만 사용할 수 있어요.
    + 가동상태조회/상세 페이지에서 단말기에 서비스 코인 등록 기능이 개발될 예정이에요.
        + 서비스 코인은 단말기를 통한 결제 이후 기기 오동작으로 상품이 미투출 또는 미동작 시에\
        사용자에게 직전 결제 금액만큼 다시 무상으로 제공하기 위해 개발되는 기능이에요.

    ## ✔️TMS 업데이트
    ### 1. TMS 폐기파일 다운로드 시 알림기능 추가
    펌웨어 등록관리 페이지에서 폐기된 펌웨어의 파일을 다운로드 시 알림 기능이 추가되었어요.
    
    ### 2. 파일이름 대소문자 구분없이 검색기능 추가
  TMS에서 파일명을 대소문자 구분없이 검색이 가능하도록 기능이 추가되었어요.
    
    ### 3. 단말기 배포이력 펌웨어 페이지 추가
    단말기 상세정보 페이지 내부에 해당 단말기가 배포되도록 등록된 이력을 볼 수 있는 페이지가 추가되었어요.
    
    ### 4. 서비스카드 일괄등록 기능 권한 제한
    이제부터 서비스카드 일괄등록 기능은 관리자 권한에 한해서 제공됩니다.
