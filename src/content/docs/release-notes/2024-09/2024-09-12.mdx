    ## ✔️VMMS 정기 업데이트
    VMMS 의 정기 업데이트가 진행되었어요.\
    이번 주 업데이트 된 내역은 아래와 같습니다.

    ### 1. 조직루트 조회 버그 수정
    + 본인 조직 이외의 조직이 보여지는 현상 수정
        + 조직이 타 조직으로 이동되어도 기존 계정에 속할 경우 조회가 가능했던 부분을 수정했어요.
        + 앞으로는 계정이 속한 조직의 하위 조직루트가 아니면 조회할 수 없습니다.

    ### 2. 세션 유지 방식 변경
    + 세션의 유지 방식이 기존 `자동 로그아웃`에서 `시간 내 동작 시 세션 시간 연장`으로 변경되었어요.
        + 기존에는 일정 시간 동안 사용자의 동작이 없을 경우 자동 로그아웃 되었는데, 이제는 사용자의 동작이 있을 경우 세션 시간이 연장됩니다.
        + 이를 통해 사용자가 더 편리하게 서비스를 이용할 수 있도록 개선했어요.

    ### 3. 거래내역 조회 기간 달력 UI 변경
    <Figure src={Image1} caption={"거래내역 페이지"} width={700}/>
    + 조회 기간 검색 내 달력 UI 가 변경되었어요.
        + 기존에는 달력 내 일부 이미지가 보이지 않거나 미래 일자에 대한 선택이 가능한 문제가 있었어요.\
        이런 부분을 개선하고, 보다 더 사용하기 쉽고 보기 편한 새로운 달력 UI 로 변경했어요.
        + 나머지 페이지들도 순차적으로 업데이트 할 계획이에요.

    ### 4. 기타 버그 수정
    + 로딩 관련 기타 버그 수정 사항이 있어요.
    + 미사용 스타일 CSS 가 제거되었어요.
    + 조회 기간에 입력 가능한 자릿수 제한을 추가했어요.
