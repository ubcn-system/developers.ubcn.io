    ### ✔️ 캘린더 매출 표기 단위를 '만' 단위로 변경
    * 일자에 모든 금액을 표기하던 기존 캘린더에서 금액을 '만' 단위로 변경하였어요. <br/>
    매출액이 클 경우 모바일 화면에서 숫자가 너무 작아지던 문제를 해결하기 위한 변경점이에요.<br/>
    <strong>단, 매출액이 1만원 이하일 경우 `0만`으로 표기될 수 있어요.</strong>

    ### ✔️ 설치위치 검색조건을 단말기명 검색 조건으로 변경
    * `설치위치` 검색 조건을 `단말기명` 검색 조건으로 변경하였어요. <br/>
