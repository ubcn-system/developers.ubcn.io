## VMMS 업데이트

<Frame>

    ### 사용자 관리 기능 개선

    관리자의 계정 관리 편의성을 위해 다음 기능이 추가되었습니다.

    - **사용자 관리 화면에서 해당 사용자 계정으로 로그인 기능 추가** - 관리자 계정에서 **비밀번호 초기화 시 초기화 비밀번호 미리보기 지원**

    또한 비밀번호 초기화 시 다음 방식으로 사용자에게 안내됩니다.

    - 신규 비밀번호 생성 후 **등록된 인증 정보(알림톡 / 이메일)로 비밀번호 전송**

</Frame>

<Frame>

    ### 비밀번호 찾기 기능 추가

    사용자가 직접 **비밀번호를 초기화할 수 있는 기능**이 추가되었습니다.

    인증 정보를 통해 계정을 확인하며, 동일한 인증 정보로 **여러 계정이 존재할 경우 계정 목록이 함께 표시**됩니다.

</Frame>

<Frame>

    ### 사용자 정보 수정 기능 개선

    사용자 정보 수정 화면에서 다음 기능이 추가되었습니다.

    - **비밀번호 보기 기능 추가**

    비밀번호 입력 시 사용자가 내용을 확인할 수 있도록 지원합니다.

</Frame>

<Frame>

    ### 자주묻는질문 기능 개선

    자주묻는질문(FAQ) 게시글에 **최종 수정일 표시 기능이 추가**되었습니다.

</Frame>

<Frame>

    ### 상품관리 설명 개선

    기존 **운영 머신기 관리 > 상품관리** 메뉴의 설명이 추가되었습니다.

    #### 상품 등록 유효성 검증 추가

    상품 등록 시 다음 항목에 대한 검증이 추가되었습니다.

    - 중복 컬럼 등록 방지 - 컬럼번호 미등록 방지

    #### 표시 항목 변경

    상품 및 거래 데이터의 명칭이 다음과 같이 정리되었습니다.

    - 운영 머신기 관리  - `등록가`, `가격` → **원가**

    - 상품관리 - `등록가`, `가격` → **원가**

    - 거래내역 - `금액`, `결제총금액`, `거래금액` → **판매가** - `취소금액` → **취소액**

    또한 상품 관리 화면에서 **원가 표시 항목이 제거**되었습니다.

</Frame>
