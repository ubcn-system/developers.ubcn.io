    ## ✔️VMMS 정기 업데이트
    VMMS 의 정기 업데이트가 진행되었어요.\

    ### 1. 운영 자판기 관리
    <Figure src={Image1} caption={"수정이력, 거래이동 버튼 추가"}/>
    + 상세페이지에서 `수정이력` 버튼을 위로 이동시키고, `거래이동` 버튼을 추가했어요.
    <Figure src={Image2} caption={"거래이동 기능 추가"}/>
    + 단말기의 `조직루트`, `담당자`, `설치위치` 가 변경되며 기존 거래를 보지 못하는 경우가 존재했어요\
    이를 해결하기 위해 `거래이동` 기능을 추가했어요.
    + 현재 등록된 정보로 거래를 이동할 수 있어요.
    <br/>
    + 운영 자판기 관리 페이지에서 엑셀 저장 시, 임시로 최대 3,000건 까지만 지원하도록 변경되었어요.\
    (3,000건 이상의 데이터는 시스템사업부(system@ubcn.co.kr) 로 문의해주세요.)
    <br/>
</Frame>
