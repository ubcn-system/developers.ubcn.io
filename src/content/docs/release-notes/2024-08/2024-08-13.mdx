    ## ✔️결제 유형 추가
    <Figure src={Image1} caption={"신규 결제 유형 추가"} width={200}/>
    신규 결제 유형이 추가되어, 지원하는 결제 유형이 확장되었어요. VMMS 로의 병합은 검토 중이기 때문에 특정 사이트에서만 조회 가능한 결제 유형도 있지만요.
    <br/>
    이와 함께 더 이상 사용하지 않는 일부 결제 유형은 제거하였어요.
    <br/>
    `정보수집(재료잔량)`, `정보수집(잔여수량)`,  `기타결제(무상판매)`, `VEMO(카드)`, `VEMO(쿠폰)`, `VEMO(이벤트)` 결제 유형이 신규로 추가되었어요.
    <br/>
    추후 단말기에서의 업데이트를 통해 더욱 다양한 결제 유형을 지원할 예정입니다.
    <br/>
    ### 💸지원하는 결제 유형
    + 신용
    + 현금, 현금영수증(가맹점의 국세청 자동 신고)
    + 선불거래(티머니, 캐시비, 레일플러스 등)
    + 간편결제(카카오페이, 네이버페이, 페이코 등)
    + 단말기 정보
    + 성인인증
    + 쿠폰
    + 멤버십 포인트
    + 사원카드
    + 정보수집(장바구니 품목, 재료잔량, 잔여수량)
    + 기타결제(쿠폰, VEMO(카드), VEMO(쿠폰), VEMO(이벤트), 무상판매)
