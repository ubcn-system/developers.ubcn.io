    ### ✔️VMMS 정기 업데이트
    VMMS 의 정기 업데이트가 진행되었어요.\
    이번 주 업데이트 된 내역은 아래와 같습니다.
    <br/>

    ##### 1. 거래내역 신규 검색 조건 추가
    <Figure src={Image} caption={"신규 검색 조건"}/>
    거래내역 페이지에 신규 검색 조건이 추가되었어요.\
    `카드번호`, `승인번호` 를 검색 조건으로 추가하여 더욱 편리하게 검색할 수 있습니다.\
    단, 기간 내 범위 조회가 아닌 `당일`에 한하여 조회가 가능합니다.

    ##### 2. 취소 가능 시간 알림 추가
    <Figure src={Image2} caption={"취소 가능 시간 알림"}/>
    취소 시, '취소 가능한 시간대 알림'이 추가되었어요.<br/>\
    `매입` 이 진행 중일 때 취소하는 경우, 별도의 알림 없이 취소가 되지 않아 PG사, VAN사 또는 카드사로 문의를 해야하는 불편함을 겪는 경우가 있어 안내 문구가 추가되었습니다.
    <br/>
    `매입` 이 진행 중인 <b>매일 새벽 00시 - 05시</b> 사이에는 모든 `취소`가 불가능해요.
    <br/>
    + 취소 불가능 시간: 00:00:00 ~ 04:59:59
    + 취소 가능 시간: 05:00:00 ~ 23:59:59
    ##### 3. 거래내역 취소 기능 개선 및 버그 수정
    <Figure src={Image3} caption={"거래 상태에 따른 취소 버튼 표기 변경"}/>
    + '거래 진행상태' 에 따라 취소 문구가 변경돼요.
        + 기존 `거래취소` 버튼명이 세부 문구로 변경되어 정확한 취소 내용을 확인할 수 있습니다.
            + `승인취소`, `매입취소`, `거래삭제`
    + 실제 PG 또는 VAN사로 `매입`이 진행되었으나 거래의 진행상태가 업데이트 되지 않고 `구매`로 남아있는 경우(배치가 진행 중인 경우), `승인취소` 로 진행되는 버그가 수정되었어요.
    + 취소 기능 제한 설정 추가
        + 특정 소속에 한하여 마감 여부에 따라 모든 `취소`가 불가능하도록 설정할 수 있게 되었어요. (`코카콜라`)
