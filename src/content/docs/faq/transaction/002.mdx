## ❓ 승인 실패가 반복되는 이유는 무엇인가요?

<Frame>

승인 실패는 카드사, 통신, 단말기 설정 등 여러 요인에 의해 발생할 수 있어요.

일회성 오류가 아닌 반복적인 승인 실패는 다음과 같은 원인을 의심해볼 수 있어요.

<br/>

### ✔ 카드 또는 결제수단 문제

- 한도 초과
- 카드 정지 또는 유효기간 만료
- 발급사 정책에 따른 승인 거절

<br/>

### ✔ 통신 문제

- 단말기의 인터넷 (모뎀 또는 이더넷) 연결 이상
    - 이더넷의 경우 본사 네트워크 정책 보유 시, 방화벽 INBOUND 확인 필요
    - <a href={"https://ubcn.notion.site/UBCn-Network-Pay-21930b3d33e24caea25d97d3a314f066?pvs=4"} target={"_blank"}>네트워크 바로가기</a>
- 전자결제 서비스 회사와의 통신 오류

<br/>

### ✔ 가맹 문제

- 카드사 미가맹점
- 미등록단말기 상점

<br/>

### ✔ 단말기 설정 문제

- 가맹 정보 오류
- 펌웨어 버전 불일치
- 내부 설정값 이상

반복적으로 발생하는 경우, 가맹이 정상적으로 되었는지 확인 후, **통신 상태와 단말기 설정을 우선 점검하는 것이 좋아요.**

</Frame>
