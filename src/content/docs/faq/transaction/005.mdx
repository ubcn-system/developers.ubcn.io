## ❓ 지역화폐 거래가 가능한가요?

<Frame>

지역화폐 사용 가능 여부는 **가맹 형태와 카드사 계약 여부에 따라 달라져요.**

카드 단말기에서는
**선불형을 제외한 카드형 지역화폐(IC칩 기반)** 만 사용이 가능해요.

<br/>

### ✔ VAN 가맹점의 경우

일부 카드형 지역화폐는
카드사 가맹 계약 및 지역화폐 가맹 등록이 완료되어 있다면 사용이 가능해요.

지역화폐는 일반 신용·체크카드와 달리
지자체 정책 및 카드사 계약 조건에 따라 사용 가능 업종과 가맹점이 제한될 수 있어요.

따라서 카드사와의 지역화폐 가맹 등록이 되어 있어야 결제가 가능해요.

<br/>

### ✔ PG 가맹점의 경우

일반적으로 PG 가맹 구조에서는
지역화폐 결제가 제한되는 경우가 많아요.

가맹점에서 지역화폐 운영사 또는 카드사를 통해
직접 가맹 등록 여부를 확인해야 해요.

이는 카드사 직접 가맹 여부 및 지역화폐 정책에 따른 차이예요.

<br/>

### ✔ 사용 가능 여부 확인 방법

- 카드사 가맹 상태 확인
- 지역화폐 가맹 등록 여부 확인
- 업종 제한 여부 확인

특이사항이 없다면,
가맹점에서 지역화폐 가맹 신청 후 약 3~4일 이후부터 결제가 가능해요.

<br/>

---

<br/>

### ✔ 대표적인 카드형(IC카드) 지역화폐

국내에는 다음과 같은 카드형 지역화폐가 운영되고 있어요.

<br/>

**서울 – 서울페이+ (서울사랑상품권 카드형)**
고객센터: 1600-6120

**인천 – 인천e음 (이음카드)**
고객센터: 1811-8668

**부산 – 동백전 (카드형)**
고객센터: 1577-1432

**대구 – 행복페이 (카드형)**
고객센터: 1566-5050

**경기 – 경기지역화폐 (카드형)**
고객센터: 1899-7997

<br/>

지역화폐 결제 가능 여부는
가맹 형태(VAN / PG) 및 카드사 계약 조건에 따라 달라질 수 있으므로,
정확한 확인은 해당 지역화폐 고객센터 또는 카드사에 문의하는 것이 좋아요.
