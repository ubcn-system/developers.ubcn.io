## ❓ 승인됐는데 상품이 나오지 않았어요

<Frame>
대표적으로 승인 과정에서 일어나는 **카드사의 통신장애**가 있고,
승인 이후에는 단말기에서 머신기로 전달되는 **투출 신호가 정상적으로 전달되지 않았거나 머신기 문제로 인해 상품이 나오지 않을 수 있어요.**
<br/>

승인과 상품 투출은 다음과 같은 흐름으로 이루어져요.

`승인 완료` → `단말기 승인 결과 수신` → `머신기에 투출 신호 전달` → `상품 투출`

이 과정 중 어느 한 단계에서 문제가 발생하면 상품이 나오지 않을 수 있어요.

<br/>

### ✔ 발생 가능한 원인

- 카드사 통신 장애
- 머신기 내부 모터 또는 기구 이상
- 투출 센서 오류
- 단말기와 머신기 간 신호 전달 문제
- 일시적인 통신 오류

<br/>

### ✔ 확인 방법

- 해당 거래의 승인 상태 확인(최종 승인 여부)
    - 전산에서는 `망상취소` 로 보이지만, 카드사 시스템이 정상화 된 후 카드사 재량으로 대행승인이 진행되었을 수 있어요.
    - 이 경우, 실제 거래 파악을 위해 고객센터를 통해서만 취소가 가능합니다.
        - 취소 시 필요한 정보: **거래일자, 거래시간, 카드번호, 승인번호, 승인금액**
- 머신기 점검
    - 머신기 품절이나 고장이 발생했을 수 있어요.
- 동일 위치에서 반복 발생 여부 확인

    <br/>
상품 미투출 문제는 대부분 **머신기 점검이 필요한 경우**가 많기 때문에 점검이 필요해요.

</Frame>
