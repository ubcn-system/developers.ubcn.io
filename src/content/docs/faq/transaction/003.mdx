## ❓ 승인취소와 매입취소의 차이는 무엇인가요?

<Frame>

승인취소와 매입취소는 **취소가 이루어지는 시점**이 달라요.

<br/>

### ✔ 승인취소

- 승인 직후, **마감 전에 취소하는 경우**
- 카드 사용자의 한도 차감이 원상복구돼요
- 체크카드의 경우 출금된 금액이 즉시 반환돼요
- 거래가 매입 단계로 넘어가지 않아요

즉, **승인이 완료된 `구매` 상태의 거래를 즉시 되돌리는 취소**예요.

<br/>

### ✔ 매입취소

- 이미 **마감이 완료되고 매입이 진행된 이후** 취소하는 경우
- 정산 대상에 포함된 거래가 취소돼요
- 다음 정산 시 해당 금액이 차감되어 입금될 수 있어요

즉, **정산 대상에 포함된 거래를 차감하여 되돌리는 취소**예요.

<br/>

취소 시점에 따라 마감·매입 여부와 정산 금액 반영 방식이 달라질 수 있어요.

<br/>

### ✔ 공통점

취소는 카드번호, 승인번호, 거래금액 등 **거래 시 생성된 정보를 기반으로 처리**돼요.

카드번호는 관련 법령에 따라 **최대 3개월까지만 보관 가능**하며,
이 기간이 지나면 카드번호는 마스킹 처리돼요.

따라서 거래일로부터 3개월이 지난 이후에는
유비씨엔 전산에서 직접 취소 처리가 어려울 수 있어요.

이 경우에는 **PG사, VAN사 또는 카드사를 통해 별도 절차로 처리**해야 해요.


</Frame>
