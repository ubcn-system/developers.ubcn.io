## ❓ 망상취소는 어떤 경우에 발생하나요?

<Frame>

망상취소는 승인 요청 후 카드사와의 통신 과정에서 **정상적인 승인 응답을 받지 못했을 때 자동으로 발생하는 취소**예요.

일반적으로 다음과 같은 상황에서 발생할 수 있어요.

<br/>

### ✔ 통신 중단 또는 지연

- 승인 응답 수신 전 네트워크 단절
- 카드사 응답 지연
- 전자결제 서비스 회사 통신 오류

<br/>

이 경우 단말기나 전산은 거래를 확정할 수 없기 때문에, 이중 결제 또는 오결제를 방지하기 위해 자동으로 취소 처리해요.

취소 처리한 결과를 머신기로 전달해서 서비스 제공을 하지 않도록 하고 있어요.

망상취소는 **안전장치에 가까운 자동 취소**라고 이해하면 돼요.

</Frame>
