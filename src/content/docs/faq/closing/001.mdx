## ❓ 결제가 된 상태인데 아직 매입이 안 된 이유는 무엇인가요?

결제 후 서비스 제공 직후의 상태인 `구매` 상태는 **승인이 정상적으로 완료된 상태**를 의미해요.
하지만 구매 상태라고 해서 즉시 매입이 진행되는 것은 아니에요.

매입은 **마감이 완료된 이후에 진행**돼요.

<br/>

### ✔ 마감이 완료되지 않은 경우

거래가 시스템에서 아직 마감되지 않았다면 매입 단계로 넘어가지 않아요.
이 경우 전산상 상태는 구매로 유지돼요.

특히 단말기 기준의 **수동마감 설정을 사용하던 경우**,
상점에서 마감을 직접 진행하지 않으면 매입이 요청되지 않아요.

<br/>

### ✔ 정리

- 구매 = 승인 완료 상태
- 매입은 마감 이후 단계
- 마감이 되지 않으면 매입은 진행되지 않음

구매 상태가 지속되는 경우, 해당 거래의 **마감 여부**를 먼저 확인하는 것이 좋아요.
