## ❓ 상태정보 전송 주기는 어떻게 되나요?

<Frame>

단말기는 머신기로부터 상태정보를 수집한 뒤,
**일정 주기(일반적으로 약 1시간 간격)**로 유비씨엔 서버에 전송해요.

이 전송은 자동으로 이루어지며, 사용자가 직접 실행하지 않아요.

<br/>

### ✔ 전송 과정

1. 단말기가 머신기로 신호 전송
2. 머신기 상태값 수신
3. 상태정보를 서버로 업로드
4. 웹 화면에 반영

<br/>

### ✔ 전송 중 단말기 동작

상태정보를 서버로 전송하는 동안에는
단말기가 잠시 멈출 수 있어요.

이 시간 동안에는 결제가 일시적으로 진행되지 않을 수 있지만,
전송이 완료되면 정상적으로 동작해요.

</Frame>
