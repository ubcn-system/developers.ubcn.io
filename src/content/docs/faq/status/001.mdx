## ❓ 머신기의 실제 상태와 웹에 표시되는 상태정보가 다른 이유는 무엇인가요?

<Frame>

가동상태조회는 단말기가 **정해진 주기마다 머신기로 신호를 보내 상태정보를 수신한 뒤 서버로 업로드하는 방식**으로 동작해요.

이 과정에서 머신기의 실제 상태와 웹에 표시되는 상태정보가 다르게 보일 수 있어요.

이는 대부분 **머신기 또는 단말기 설정·버전 차이**로 인해 발생해요.

<br/>

### ✔ 발생 가능한 주요 원인

1. **머신기 이상으로 인한 제3의 오류**
   내부 센서, 통신 모듈, 신호 회로 이상 등

2. **머신기 AS 점검 이후 설정값 변경 또는 틀어짐**

3. **머신기 펌웨어 버전과 단말기 통신 규격이 맞지 않는 경우**

4. **머신기 설정 오류**

5. **단말기 펌웨어 버전 차이**

6. **단말기 설정 오류**

<br/>

상태정보는 단말기가 머신기로부터 받은 값을 그대로 서버에 전달하는 구조이기 때문에,
장비 간 통신 규약이나 설정이 일치하지 않으면 실제 상태와 다르게 표시될 수 있어요.

<br/>

## ✔ 해결 방법

근본적인 해결을 위해서는 **머신기 제조사 또는 관리 업체를 통해 머신기 점검을 진행**해야 해요.

- 머신기 설정값 확인
- 펌웨어 버전 점검
- 통신 규격 일치 여부 확인
- 하드웨어 이상 여부 점검

단말기와 서버는 머신기가 전달한 값을 기반으로 동작하기 때문에,
장비 측 점검이 우선적으로 필요해요.
