## 연동 / API란?

`연동 / API`는 **외부 시스템이나 내부 서비스가 유비씨엔 전산과 데이터를 주고받기 위한 인터페이스**를 의미해요.

API(Application Programming Interface)를 통해 시스템 간 자동 연동이 이루어져요.

</Frame>

<Frame>

## 연동 대상

유비씨엔 전산은 다음과 같은 대상과 연동돼요.<br/><br/>

- 전자결제 서비스 회사(PG / VAN / 간편결제사)
- 카드사
- 내부 운영 시스템
- 외부 관리 시스템

</Frame>

<Frame>

## 연동되는 주요 데이터

연동을 통해 다음과 같은 데이터가 오가요.<br/><br/>

- 승인 요청 및 승인 결과
- 거래 상태 정보
- 매입·정산 관련 데이터
- 상태정보 및 운영 데이터

</Frame>

<Frame>

## API와 결제 처리

결제 승인, 매입 요청, 상태정보 수집 등 주요 전산 처리는 API를 통해 이루어져요.

API 통신이 정상적으로 이루어지지 않으면 승인 실패, 데이터 누락 등이 발생할 수 있어요.

</Frame>

<Frame>

## 연동 방식의 특징

API 연동은 자동화되어 있으며, 사람이 직접 개입하지 않아요.

보안, 인증, 데이터 무결성이 중요하며, 모든 요청과 응답은 전산에 기록돼요.

</Frame>

<Frame>

## 정리

- 연동 / API는 시스템 간 데이터 통신 방식
- 결제·운영 전산의 핵심 기반
- 승인, 매입, 상태정보 처리에 사용
- 안정성과 보안이 중요

</Frame>
