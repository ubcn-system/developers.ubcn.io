## 배치란?

`배치`는 **실시간이 아닌, 정해진 시점이나 주기에 따라 자동으로 실행되는 전산 처리 작업**을 의미해요.

유비씨엔 전산에서는 대량 데이터 처리나 정기 작업을 배치 방식으로 수행해요.

</Frame>

<Frame>

## 배치가 필요한 이유

모든 작업을 실시간으로 처리하면 시스템 부하가 커질 수 있어요.

배치는 다음과 같은 작업에 사용돼요.<br/><br/>

- 거래 데이터 정리
- 매입·정산 대상 데이터 생성
- 상태정보 집계
- 통계 및 리포트 생성
- 로그 정리 및 보관

</Frame>

<Frame>

## 배치와 결제 흐름의 관계

배치는 승인처럼 즉시 처리되는 단계가 아니에요.

결제가 완료된 이후, **후처리 과정**에서 배치가 동작해요.
예를 들어, 마감 이후 매입 대상 거래를 생성하는 작업은 배치로 처리돼요.

</Frame>

<Frame>

## 배치 실행 방식

배치는 다음 방식으로 실행돼요.<br/><br/>

- 일정 시간마다 자동 실행
- 하루 1회 또는 다회 실행
- 특정 조건 충족 시 실행

대부분의 배치는 사용자가 직접 실행하지 않아요.

</Frame>

<Frame>

## 배치와 사용자 영향

배치는 백그라운드에서 실행되며, **일반적인 결제나 조회에는 영향을 주지 않아요.**

다만 배치 지연이나 실패 시, 매입·정산 반영이 늦어질 수 있어요.

</Frame>

<Frame>

## 정리

- 배치는 비실시간 자동 처리 작업
- 대량 데이터·후처리 작업에 사용
- 결제 완료 이후 단계에서 동작
- 사용자 개입 없이 자동 실행

</Frame>
