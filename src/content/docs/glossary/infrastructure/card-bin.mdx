## 카드 BIN이란?

`카드 BIN`은 **Bank Identification Number**의 약자로, 카드 번호의 앞 6~8자리를 의미해요.

BIN에는 카드 발급사, 카드 종류, 발급 국가 등의 정보가 포함돼 있어요.

</Frame>

<Frame>

## BIN의 역할

카드 결제 시 BIN은 다음과 같은 용도로 사용돼요.<br/><br/>

- 카드 발급사 식별
- 카드 종류 구분 (신용 / 체크 / 선불 등)
- 국내 카드 / 해외 카드 구분
- 승인 요청을 전달할 카드사 결정

단말기나 전산은 BIN 정보를 기반으로 승인 요청을 어디로 보낼지 판단해요.

</Frame>

<Frame>

## BIN과 승인 과정

결제가 시도되면 단말기 또는 전산은 카드 번호에서 BIN을 추출해요. 이 정보를 통해 **어떤 카드사에 승인 요청을 보낼지**를 결정해요.

BIN 정보가 올바르지 않으면 승인 실패가 발생할 수 있어요.

</Frame>

<Frame>

## BIN과 전산 처리

전산에서는 BIN 정보를 활용해 카드 유형별 통계, 승인 성공률 분석, 이상 거래 감지 등을 수행해요.

BIN 자체는 결제 금액이나 정산과는 직접적인 관련은 없어요.

</Frame>

<Frame>

## 정리

- BIN은 카드 번호 앞 6~8자리
- 카드 발급사·종류·국가 식별에 사용
- 승인 요청 경로를 결정하는 기준 정보
- 결제 인프라의 기본 식별 정보

</Frame>
