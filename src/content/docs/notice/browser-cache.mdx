# <strong>브라우저 캐시 비우기 및 강력 새로고침 활용하기</strong>
사이트 갱신 후, 세션이 만료되거나 적용되지 않는 때가 있습니다.
위 문제는 대개 웹 캐시로 인해 생기는 문제로, 사용 중인 웹 브라우저의 `캐시를 초기화` 하는 것으로 해결할 수 있습니다.
<br/>
<Aside type="danger" title={"주의"}>일반 캐시 초기화 방법의 경우 웹 브라우저에 저장된 계정 정보, 비밀번호 등
웹사이트 설정도 함께 삭제될 수 있어, 사용자에 따라 매우 곤란해질 수 있습니다.</Aside>
<br/>
만약 Google 크롬 브라우저를 사용하는 경우 `캐시 비우기` 및 `강력 새로고침`을 사용해 볼 수 있습니다.
이 기능은 웹 브라우저 전체 설정을 삭제하지 않고, <strong>원하는 페이지의 캐시만 간편하게 초기화</strong>하는 것이 가능합니다.
<br/>
크롬엔진을 사용하는 모든 브라우저(chrome, whale, safari, opera, firefox, MS Edge, Brave 등)에 해당합니다.
### <strong>사용방법 1.(권장)</strong>
<Block>
<Steps>
1. 내 PC에서 Google 크롬 브라우저를 켭니다.
2. 반영되지 않거나 동작하지 않는 메뉴(페이지)에 접속합니다.
3. PC 환경인 경우 키보드에서 아래 단축키를 누릅니다. (모바일의 경우 4번으로 가세요.)<br/>
        1. ⓘ 예시 - Windows: `Ctrl` + `Shift` + `R`
        2. ⓘ 예시 - Mac: `Command` + `Shift` + `R` 을 동시에 누릅니다.
4. 화면을 최상단으로 이동하여 손가락 한 개 또는 두 개를 이용하여 화면 쓸어내리기를 진행합니다.
    (새로고침이 진행되는지 확인합니다.)
</Steps>
    {<Figure src={Image02} caption={"쓸어내리기"} width={300}/>}
</Block>
### <strong>사용방법 2.</strong>
<Block>
<Steps>
1. 내 PC에서 `Google 크롬 브라우저`를 켭니다.
2. 반영되지 않거나 동작하지 않는 메뉴(페이지)에 접속합니다.
3. 크롬 브라우저 오른쪽 상단 더보기 아이콘(점 3개)을 클릭합니다.
4. [도구 더보기 > 개발자 도구]를 클릭해 개발자 도구를 열어줍니다. 또는 아래의 단축키를 사용할 수도 있습니다.<br/>
    1. ⓘ 예시 - Windows: `F12`
    2. ⓘ 예시 - Mac: `Command` + `Option` + `I`

5. 오른쪽에 개발자 도구가 나타나면 크롬 브라우저 왼쪽 상단 새로고침 아이콘에 마우스 포인터를 올립니다.
6. 마우스 오른쪽 버튼을 클릭합니다.
7. 다음 그림처럼 새로고침 메뉴가 나타나면 `캐시 비우기 및 강력 새로고침` 항목을 클릭합니다.
8. 캐시 비우기 및 강력 새로고침 과정이 완료되면, 페이지 내 콘텐츠가 최근 변경한 것들로 적용되었는지 확인합니다.
</Steps>
{<Figure src={Image01} caption={"캐시 비우기"} width={300}/>}
</Block>


### <strong>사용방법 3. (삼성인터넷)</strong>
<Block>
<Steps>
1. 하단 선 세 개 더보기 메뉴 클릭
2. 인터넷 사용 기록 클릭
3. 인터넷 사용 기록 삭제
4. 쿠키 및 사이트 데이터, 캐시 이미지와 파일 체크
5. 데이터 삭제
</Steps>
