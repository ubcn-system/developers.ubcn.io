# <strong>카카오 브라우저가 아닌 다른 브라우저로 접속하기</strong>
카카오톡에서 전달받은 URL(접속링크)로 VMMS 접속 시 로그인이 불가한 사항이 있습니다. <br/>
위 문제는 권장하는 브라우저가 아닌 다른 브라우저에서 VMMS 접속 시 발생할 수 있는 문제로,
크롬, 삼성인터넷, 사파리 등의 권장하는 브라우저 사용을 권장드립니다. <br/>
아래는 카카오 브라우저에서 다른 브라우저로 접속하는 방법을 서술합니다.
